# 변경 로그: 초보자를 위한 MCP 커리큘럼

이 문서는 초보자를 위한 Model Context Protocol (MCP) 커리큘럼에 이루어진 주요 변경 사항을 기록한 문서입니다. 변경 사항은 최신 순으로 기록됩니다.
